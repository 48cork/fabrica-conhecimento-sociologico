# fabrica-de-conhecimento-sociologico

> Squad de agentes sociológicos para criar aulas criativas e analisar a sociedade de plataformas

## Teóricos

| Agente | Teórico | Perspectiva |
|--------|---------|-------------|
| `@weber` | Max Weber | Racionalização, burocracia, ação social |
| `@marx` | Karl Marx | Capitalismo, alienação, luta de classes |
| `@durkheim` | Émile Durkheim | Solidariedade social, fatos sociais, anomia |
| `@bourdieu` | Pierre Bourdieu | Capital, campo, habitus |
| `@ghiraldelli` | Paulo Ghiraldelli | Filosofia da educação, pragmatismo |

## Uso

```
@squad-creator *activate fabrica-de-conhecimento-sociologico
```
